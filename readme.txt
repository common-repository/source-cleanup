Clean up your source code by removing WP version, shortlink from WP head and moving javascript files to the footer.

== Description ==

This plugin removes the WP version and shortlink from WP head and, moves javascript to the footer for better performance. The WordPress version number appended to your CSS and script files is also removed by this plugin.

== Installation ==

= Wordpress Installer = 
1. After you have downloaded your file, log into your WordPress Admin Section.
1. Go to Plugins-> Add New.
1. Go to the Upload Link.
1. Navigate to your file.
1. Click on the Upload Button.
1. After it is has installed, click on the Activation Link.
